# Pocket Shadow Cat Desktop Pet

This is a tiny desktop-pet prototype inspired by the soft feeling of black-cat
anime mascots. The design is original: a round black cat with aqua eyes,
idle motions, click reactions, a right-click menu, and a transparent always-on-top
window.

## Run

```powershell
python desktop_pet/black_cat_pet.py
```

## Controls

- Drag with the left mouse button to move the pet.
- Click to trigger random reactions.
- Right-click to open the menu.
- Use `Toggle stroll` to turn slow self-walking on or off.
- Press `Esc` or choose `Exit` to close it.

## Behaviors

- Idle breathing and bobbing.
- Random blinks.
- Tail swishes.
- Ear twitches.
- Stretching.
- Napping.
- Excited, shy, snack, and pounce reactions.
- Optional slow strolling across the desktop.

No third-party packages are required.
